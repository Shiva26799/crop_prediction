# 🌾 Crop Prediction System

🧠 Backend (Flask)
Navigate to backend:

cd backend
Install dependencies:

pip install flask flask-cors torch numpy pandas scikit-learn
Run Flask API:

python app.py

💻 Frontend (React)
Navigate to frontend folder:

cd frontend-agri/crop-prediction
Install dependencies:

npm install
Run development server:

npm run dev
